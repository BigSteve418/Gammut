Play the game by running "gammut.py"

Option definitions
    Start/join a local multiplayer game: Choose if you want to use local multiplayer
    Start game as Host: Choose if you want your computer to act as the server (requires your firewall to permit incoming traffic)
    Address: Enter the local address provided by the host in order to connect to a multiplayer game
    Enter board size: Choose the board size/layout to play on
    Stream moves from file: Play back recorded moves from a game, does not work with multiplayer
    Record moves to file: Record the moves that you make in the game to a file fore playback later (works with multiplayer)
    
See "How to play.txt" for gameplay instructions
